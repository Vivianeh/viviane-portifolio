PROJETO 1:

CURSO DE DESENHO ONLINE

O primeiro projeto que desenvolvi foi um site de curso online de desenho, criado com o objetivo de aplicar meus conhecimentos iniciais em HTML e CSS. Embora não tenha sido publicado, foi uma experiência fundamental para colocar em prática o que eu havia aprendido. Esse exercício não só me ajudou a entender melhor os conceitos de desenvolvimento web, como também estimulou minha criatividade.

PROJETO 2:

BLOG SOBRE O MEIO AMBIENTE

O segundo projeto foi a estrutura de um blog, no qual escolhi como tema a preservação ambiental. Nele, tive a oportunidade de explorar ainda mais o uso de HTML e CSS, experimentando com layout e organização de conteúdo. Apesar de ser um projeto pessoal e não publicado, foi uma excelente prática que contribuiu muito para o meu desenvolvimento e entendimento da construção de páginas web.

PROJETO 3:

FORMULÁRIO

O terceiro projeto consistiu em uma página para preenchimento de formulário. Embora não tenha sido publicada, essa experiência me permitiu colocar em prática meus conhecimentos sobre a estrutura e funcionamento de formulários em HTML, consolidando meu aprendizado. Além de também utilizar conceitos básicos de CSS para deixar um visual mais agradável.
